# CodeToImpact

## .env content

- NODE_ENV = development
- PORT = 8000
- MONGO_URI = `mongodb uri`
- JWT_SECRET = abc123

## to run local development server

`npm install`

`npm run dev`

## to run local frontend server

`cd frontend`

`npm install`

`cd ..`

`npm run client`
